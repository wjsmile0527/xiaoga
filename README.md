# 🎮 数独小游戏

一个简洁美观的命令行数独游戏，支持随机生成、提示和彩色显示。

## 编译

```bash
g++ -o sudoku sudoku.cpp
```

## 运行

```bash
./sudoku
```

## 游戏规则

数独目标：在 9×9 的格子中填入数字 1-9，使每行、每列、每个 3×3 小宫格内都包含 1-9 且不重复。

## 操作指令

| 指令 | 说明 | 示例 |
|------|------|------|
| `n` | 开始新游戏 | 输入 `n` 重新生成新题目 |
| `h` | 获取提示 | 随机填入一个正确数字 |
| `q` | 退出游戏 | 结束程序 |
| `行 列 数字` | 填入数字 | `1 2 5` 表示第1行第2列填入5 |

## 界面说明

- **绿色数字** - 题目给出的初始数字，不可修改
- **黄色数字** - 你填入的答案
- **黑点 (·)** - 待填的空位
